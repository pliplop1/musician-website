Politique de Confidentialité

Date de dernière mise à jour :

Conformité RGPD : Cette politique de confidentialité est conforme au Règlement Général sur la Protection des Données (RGPD) de l'Union Européenne.

1. Responsable du traitement des données

Le site Duo Black & White est responsable du traitement de vos données personnelles.

Pour toute question concernant vos données, vous pouvez nous contacter via le formulaire de contact du site.

2. Données collectées

Nous collectons et traitons les données personnelles suivantes :

2.1 Données d'inscription

  • Nom d'utilisateur : pour identifier votre compte
  • Adresse e-mail : pour communiquer avec vous et récupérer votre mot de passe
  • Mot de passe : stocké de manière sécurisée (chiffré avec BCrypt)
  • Date de création du compte : pour calculer votre ancienneté

2.2 Données de profil (optionnelles)

  • Prénom et nom : pour personnaliser votre profil
  • Photo de profil : image stockée de manière sécurisée
  • Biographie : pour vous présenter (max 500 caractères)

2.3 Données d'activité

  • Concerts favoris : pour personnaliser votre expérience
  • Commentaires : vos contributions sur les concerts
  • Badges obtenus : récompenses liées à votre activité

2.4 Données de sécurité

  • Historique de connexion : date, heure, adresse IP, navigateur (conservé 30 jours)
  • Tentatives de connexion échouées : pour protéger votre compte contre les attaques
  • Tokens de réinitialisation de mot de passe : valables 1 heure uniquement

3. Finalités du traitement

Vos données sont utilisées pour :

  • Gérer votre compte utilisateur et votre authentification
  • Vous permettre d'utiliser les fonctionnalités du site (favoris, commentaires, badges)
  • Vous envoyer des e-mails de réinitialisation de mot de passe
  • Assurer la sécurité de votre compte (détection d'attaques par force brute)
  • Améliorer nos services et l'expérience utilisateur

4. Base légale du traitement

Le traitement de vos données repose sur :

  • Votre consentement : lors de votre inscription
  • L'exécution du contrat : nécessaire pour fournir nos services
  • Nos intérêts légitimes : sécurité et amélioration des services

5. Durée de conservation

Conservation des données :

  • Compte actif : tant que votre compte existe
  • Historique de connexion : 30 jours maximum
  • Tokens de réinitialisation : 1 heure (suppression automatique après expiration)
  • Après suppression de compte : toutes vos données sont définitivement supprimées

6. Vos droits (RGPD - Articles 15 à 22)

Conformément au RGPD, vous disposez des droits suivants :

6.1 Droit d'accès (Article 15)

Vous pouvez accéder à toutes vos données personnelles depuis votre profil utilisateur.

6.2 Droit de rectification (Article 16)

Vous pouvez modifier vos informations personnelles à tout moment depuis :

  • Modifier mon profil
  • Modifier mes informations détaillées

6.3 Droit à la portabilité (Article 20)

Vous pouvez télécharger toutes vos données dans un format JSON structuré :

  • Exporter mes données (JSON)

6.4 Droit à l'effacement - "Droit à l'oubli" (Article 17)

Vous pouvez supprimer définitivement votre compte et toutes vos données associées :

  • Supprimer mon compte définitivement

⚠️ Attention : La suppression est irréversible. Toutes vos données (profil, favoris, commentaires, badges, historique) seront définitivement supprimées.

6.5 Droit d'opposition (Article 21)

Vous pouvez vous opposer au traitement de vos données à des fins de marketing direct (actuellement non applicable sur ce site).

6.6 Droit de limitation (Article 18)

Vous pouvez demander la limitation du traitement de vos données dans certaines circonstances.

7. Sécurité des données

Nous mettons en œuvre des mesures de sécurité robustes pour protéger vos données :

  • Chiffrement des mots de passe : BCrypt avec salt aléatoire
  • Protection anti-brute force : blocage après 5 tentatives échouées (15 minutes)
  • Validation stricte des mots de passe : 8 caractères minimum, majuscules, minuscules, chiffres, caractères spéciaux
  • Protection CSRF : tokens anti-contrefaçon sur tous les formulaires
  • En-têtes de sécurité HTTP : CSP, X-Frame-Options, HSTS, Referrer-Policy
  • Tokens de réinitialisation sécurisés : UUID uniques, expiration 1 heure, usage unique
  • HTTPS : toutes les communications sont chiffrées (recommandé en production)

8. Partage des données

Nous ne vendons ni ne partageons vos données personnelles avec des tiers.

Vos données restent exclusivement sur nos serveurs et ne sont utilisées que pour les finalités décrites dans cette politique.

9. Cookies

Notre site utilise les cookies suivants :

  • Cookie de session (JSESSIONID) : nécessaire pour votre authentification (supprimé à la déconnexion)
  • Cookie CSRF : protection contre les attaques par contrefaçon de requêtes

Ces cookies sont essentiels au fonctionnement du site et ne nécessitent pas de consentement préalable (cookies strictement nécessaires).

10. Modifications de la politique

Nous nous réservons le droit de modifier cette politique de confidentialité à tout moment. Les modifications seront publiées sur cette page avec une nouvelle date de mise à jour.

11. Contact

Pour toute question concernant cette politique ou l'exercice de vos droits RGPD, vous pouvez :

  • Utiliser le formulaire de contact du site
  • Gérer directement vos données depuis votre profil utilisateur
